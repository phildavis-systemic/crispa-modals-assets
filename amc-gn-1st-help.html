Australian Marriage Certificate Extended Help Modals

Notes to Devs

Ideally the modal height for all states will be the height of the tallest modal (so it does not change size as you select a State or Territory).

These modals apply to the Australian Marriage Certificate document type in both the Primary Use in the Community (PCD) and Change of Name (CON) sections.

Given Names (1st Person) Modal - WA Selected

Given Names (1st Person) Help

Select the State/Territory that registered the marriage:

In this field you must enter the given names for the person who is shown in the uppermost row row, or the leftmost column, on the marriage certificate.

You must enter the given names exactly as shown (including middle names) on the certificate.

Names that use Accented Letters

If your names are shown using accented letters (e.g. ö, è, û) you must enter the non-accented version of the letter (e.g. o, e, u, etc.).

Applicants with Single Name Certificates

If your certificate only shows a single name (i.e. there are no given names or no family name), the single name can be entered in either the Given Names or Family Name fields. Leave the other field blank.

Select the State/Territory that registered the marriage:

In this field you must enter the given names for the person who is shown in the uppermost row row, or the leftmost column, on the marriage certificate.

You must enter the given names exactly as shown (including middle names) on the certificate.

Names that use Accented Letters

If your names are shown using accented letters (e.g. ö, è, û) you must enter the non-accented version of the letter (e.g. o, e, u, etc.).

Applicants with Single Name Certificates

If there are no given names shown on your certificate, you must enter exactly what is shown, as follows:

  • If the certificate is blank, leave this field blank (empty).
  • If a hyphen is shown, enter a hyphen in this field.
  • If the phrase No Registered Given Name is shown, you must enter No Registered Given Name in this field.
